# Faro Stack Advisor

## Come funziona

1. Ricevi dal wizard: tipo_progetto, tier, budget, vincoli
2. Consulta `references/recommendations.md` per la combo pre-configurata
3. Adatta in base a vincoli specifici dell'utente
4. Presenta raccomandazione con motivazione

## 3 livelli di supporto

| Livello | Significato | Cosa Faro fornisce |
|---------|-------------|-------------------|
| Tier 1 | Pattern completi | Rules dettagliate, componenti, pattern testati |
| Tier 2 | Pattern base + fetch-docs | Setup base, per il resto → documentazione ufficiale |
| Tier 3 | Guida generica | Solo consigli generali, l'utente gestisce i dettagli |
